HOW TO INSTALL:

  1. Install Half-Life from Steam, assuming you have it purchased (10-20e)
  2. Get NS1 Installer:
    • https://unknownworlds.com/ns/download-files/
    • OR
    • https://www.ensl.org/data_files/635

HOW TO RUN:

  • After install reboot steam and it should appear to your library automatically.

  • OR

  • create shortcut e.g. : "C:\Program Files\Steam\Steam.exe" -applaunch 70 -game "ns"
Have fun!

Other stuff (OPTIONAL):

  • You can install nL-pack if you want less UI clutter: http://www.ninelegends.com/releases/9l-ns-pack/
  • Custom NS1 client 3.2.1a _for linux_ can be downloaded here with some client fixes:https://github.com/Bacsu/NS
    • ONLY NEEDED FOR LINUX!!!
  • Maps and whatnot: http://www.brywright.co.uk/downloads/files/index.php?dir=natural-selection/

Basic settings:

Good launch options: -freq 144 -noforcemaccel -noforcemparms
  • Fixes mouse acceleration. WITH m_rawinput "1"
  • Should fix also "weird feeling mouse" issue!
How to achieve over 60fps:
Create userconfig.cfg in your ns-folder(E.g. C:\steamgames\steamapps\common\Half-Life\ns)
or 
just put these commands in console:

gl_vsync "0"
fps_override "1"
fps_max "145" //above 145fps makes lmg/hmg shoot faster

//"good rate settings"(dunno are they good but better than defaults):
rate "20000"
cl_updaterate "80"
cl_cmdrate "130" //5 over your fps
ex_interp "0.08"

//Mouse related stuff:
m_rawinput "1"
